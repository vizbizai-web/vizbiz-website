# VizBiz.ai

AI Visibility Intelligence for Car Dealerships

## Deployment

This site is configured for deployment on Netlify.

### Build Settings
- Build command: `npm run build`
- Publish directory: `dist`
- Node version: 20

### Features
- Static site export
- Netlify Forms integration
- Optimized for performance
- Mobile responsive
- SSL enabled

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
